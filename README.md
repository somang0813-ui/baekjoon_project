# OJ Study (Online Judge Study)

온라인 저지(백준/프로그래머스 등) 문제를 함께 풀고, 풀이를 공유/리뷰하기 위한 스터디 레포입니다.

## 목표
- 꾸준히 문제 풀이 루틴 만들기
- 서로 다른 접근(DFS/BFS/DP/그리디 등) 공유
- 코드 스타일/설명 습관 기르기

## 폴더 구조
- `problems/` : 문제별 풀이 저장
- `templates/` : 언어별 기본 템플릿
- `weekly/` : 주간 계획/회고 기록
- `RULES.md` : 파일/커밋/PR 규칙

## 사용 방법 (빠른 시작)
1. `problems/` 아래에 문제 폴더를 만든다. (예: `BOJ_1234`, `PGS_무인도여행`)
2. 문제 폴더 안에 **본인 이름 폴더**를 만든다.
3. 본인 폴더에 코드 파일을 만든다. (예: `solution.py`, `main.cpp`)
4. 커밋 후 푸시(또는 PR)한다.
